\### Здесь будут представлены команды связанные с GIT

\*\*git add\*\* - обновить файлы

\*\*git comit\*\* - добавить коммит



\##Работа с консолью

cd - \_выбрать директорию\_

ls - \_посмотреть вложения в директории\_

pwd - \_посмотреть где ты находишься\_

touch - \_создать файл\_

mkdir - \_создать папку\_

mv(dir) - \_перемешение\_

rm - \_удаление\_

\*rm -r "имя папки" - удаление папки\*
